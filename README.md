# Customer Cowork Estimator Partner App

Static GitHub Pages landing page for partner access to the Microsoft Customer Cowork Estimator workbook.

## Files

- `index.html` - partner-facing landing page
- `CustomerCoworkEstimator.xlsx` - downloadable workbook
- `.nojekyll` - keeps GitHub Pages publishing simple

## Local use

Open `index.html` in a browser and use the **Download workbook** button.

## GitHub Pages

Publish from the `main` branch root. The public app URL will be:

`https://bosh345.github.io/customer-cowork-estimator-app/`
