Model assumptions

Assumption basis

Default prompt and credit assumptions are based on Microsoft Frontier customer usage as of 5/27/2026 and assume Anthropic Opus 4.8.

  • Light: narrow context, lightweight model, 0-1 tool calls, minimal runtime.
  • Medium: richer context, capable model, several tool calls, moderate runtime.
  • Heavy: broad context aggregation, high-quality model, many tool calls, sustained runtime.
